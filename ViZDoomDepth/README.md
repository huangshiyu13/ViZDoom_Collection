ViZDoomDepth
==========
This repository is for depth prediction for ViZDoom.

## Requirement
- Ubuntu
- Python 3.6
- Tensorflow>=1.12


## Demo Preview

`python demo.py`

The script will take `images/1.png` as input and output the result to `images/_prediction.png`. The result is as below:

![](./images/1_prediction_pre.png)
