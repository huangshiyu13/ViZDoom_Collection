ViZDoomDetection
==========
This repository is for object detection for ViZDoom.

## Requirement
- Ubuntu
- Python 3.6
- Tensorflow>=1.12


## Demo Preview

`python demo.py`

The script will take `images/1.jpg` as input and output the result to `images/1_prediction.jpg`. The result is as below:

![](./images/1_prediction.jpg)
